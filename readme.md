## Hosting

This site is hosted on GitHub Pages and migrated from SquareSpace.

## Live Site

Once GitHub Pages is enabled, the site will be available at:
- `https://darylhogg.github.io/Torvity-Website/`
- Custom domain: `www.torvity.com` (after DNS configuration)

## Enabling GitHub Pages

1. Go to **Settings** → **Pages** in this repository
2. Under "Source", select the `main` branch
3. Select the root folder `/`
4. Click **Save**

## Custom Domain Setup

To use `www.torvity.com`:
1. In GitHub Pages settings, add `www.torvity.com` in the "Custom domain" field
2. Update DNS settings with your domain provider:
   - Add a CNAME record: `www` → `darylhogg.github.io`
